Carry chain

MT15 carry chain contains five modules, each one basically
a 74S182 equivalent with low_active carry signals.

Logic gates inside a module use complementary transistor logic,
so you sure could imagine what the AND/OR combinations look
like at transistor level.

Modules and ALU are wired together, like you would expect from
74181/74182, except that ALU slices are only one Bit wide...
and that is, why we need five modules.

For more details, please consult/dissect the 74S182 datasheet from Texas Instruments.
Looks like the AM2902 from AMD was compatible in functionality and pin_out.
